# Contributing

Thanks for helping improve **awesome-openclaw**.

## What belongs here

This repository is a curated list of high-quality resources for the OpenClaw ecosystem.

Good additions usually have:

- clear OpenClaw relevance
- a useful README or working documentation
- active maintenance or strong reference value
- a specific, understandable purpose

## What to avoid

Please avoid adding:

- unrelated projects that only happen to mention `claw`
- obvious duplicate forks or rename-lineage mirrors
- low-signal spammy directories
- unclear commercial landing pages with little substance
- duplicate entries across multiple sections unless there is a strong reason

## How to contribute

### Add a new project

When suggesting a new entry, try to include:

- repository link
- the best category for it
- a short factual description
- why it is relevant to OpenClaw

Entry style should follow this pattern:

```md
- [owner/repo](https://github.com/owner/repo) ![GitHub Repo stars](https://img.shields.io/github/stars/owner/repo?style=social) - Short factual description.
```

### Improve existing entries

Useful improvements include:

- fixing categorization
- shortening descriptions
- removing stale or duplicate entries
- correcting broken links
- improving consistency and formatting

## Curation rules

- Prefer the canonical repo over mirrors or duplicate forks.
- Keep descriptions short, factual, and neutral.
- Put each project in its strongest single category.
- Prefer quality over quantity.

## Pull requests

Small focused pull requests are best.

Before opening a PR, please check:

- the link works
- the project is actually relevant to OpenClaw
- the description is concise
- the entry is placed in the right section

Thanks.
